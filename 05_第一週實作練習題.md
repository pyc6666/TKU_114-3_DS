## 題目一：個人資料與 BMI 健康判斷系統

### 檔案名稱

```text
PersonalProfileApp.java
```

檔案中必須使用：

```java
public class PersonalProfileApp
```

### 題目情境

請設計一個個人健康資料系統。程式要能輸入基本資料，計算 BMI，判斷 BMI 分級，判斷是否成年，最後輸出完整報表。

這題主要整合：

- `Scanner`
- `String`、`int`、`double`、`boolean`
- 基本運算
- `if else if else`
- method 參數與回傳值

### 功能需求

程式必須輸入：

1. 姓名
2. 年齡
3. 身高，單位為公尺
4. 體重，單位為公斤

程式必須輸出：

1. 姓名
2. 年齡
3. 是否成年
4. 身高
5. 體重
6. BMI
7. BMI 分級

### 輸入驗證

必須檢查：

1. 年齡必須大於 0。
2. 身高必須大於 0。
3. 體重必須大於 0。

如果輸入不合法，必須要求重新輸入。

### BMI 分級規則

| BMI | 分級 |
|---:|---|
| 小於 18.5 | Underweight |
| 18.5 到未滿 24 | Normal |
| 24 到未滿 27 | Overweight |
| 27 以上 | Obese |

### Method 要求

至少定義以下 method：

```java
public static int readPositiveInt(Scanner sc, String message)
public static double readPositiveDouble(Scanner sc, String message)
public static double calculateBmi(double height, double weight)
public static String getBmiLevel(double bmi)
public static boolean isAdult(int age)
public static void printReport(String name, int age, boolean adult, double height, double weight, double bmi, String level)
```

### 執行範例

```text

請輸入年齡：20

請輸入體重（公斤）：55

Weight: 55.0
```

- 有輸入驗證。

## 題目二：成績統計與等第分析系統

### 檔案名稱

```text
GradeStatistics.java
```

檔案中必須使用：

```java
public class GradeStatistics
```

### 題目情境

請設計一個成績統計系統。程式可以重複輸入多筆成績，輸入 `-1` 結束。結束後輸出總分、平均、最高分、最低分、及格人數、不及格人數與平均等第。

這題主要整合：

- `while`
- 輸入驗證
- 累加器 accumulator
- 計數器 counter
- 最大值、最小值
- `if else if else`
- method

### 功能需求

程式必須：

1. 重複輸入成績。
2. 輸入 `-1` 表示結束。
3. 合法成績範圍為 0 到 100。
4. 如果輸入小於 -1 或大於 100，必須要求重新輸入。
5. 統計總分。
6. 統計平均。
7. 統計最高分。
8. 統計最低分。
9. 統計及格人數。
10. 統計不及格人數。
11. 根據平均分數輸出等第。

### 等第規則

|---:|---|
| 90 以上 | A |
| 80 到 89 | B |
| 70 到 79 | C |
| 60 到 69 | D |

如果第一個輸入就是 `-1`，代表沒有任何成績。程式必須輸出：

## 常見錯誤與診斷

| 常見錯誤 | 可能原因 | 修正方式 |
|---|---|---|
| iClass 只貼 repository 首頁 | 沒有貼到 `week1_practice` 資料夾 | 貼上 `.../tree/main/week1_practice` |
| 檔名與 class 名稱不一致 | Java 檔名沒有對應 `public class` | 讓 `PersonalProfileApp.java` 對應 `public class PersonalProfileApp` |
| 程式只在第一次操作正常 | 迴圈變數沒有更新 | 檢查 `while` 條件與選項是否重新讀取 |
| 輸入負數仍被接受 | 沒有輸入驗證 | 使用 `while` 要求重新輸入 |
| 平均計算錯誤 | 使用整數除法 | 平均使用 `double`，例如 `total / (double) count` 或 `total / 3.0` |
| method 沒有被執行 | 只定義 method，沒有呼叫 | 在 `main` 或其他 method 中呼叫它 |
| `return` 型態錯誤 | method 宣告型態與回傳值不同 | 確認 `int`、`double`、`String`、`boolean` 是否一致 |
| 選單忘記 `break` | `switch` case 繼續往下執行 | 每個需要停止的 `case` 加上 `break` |
