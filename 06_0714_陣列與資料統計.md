## 問題情境

假設要計算 5 筆成績的平均值，可以建立 5 個變數：

```java
int score1 = 80;
int score2 = 75;
int score3 = 92;
int score4 = 68;
int score5 = 88;
```

如果資料增加到 50 筆，就需要 50 個變數，而且無法方便地使用迴圈處理。陣列可以把這些資料放在同一個結構中：

```java
int[] scores = {80, 75, 92, 68, 88};
```

之後只要改變索引，就能取得不同資料。

### 概念 2：陣列宣告與建立

陣列變數必須指定元素型態。以下宣告一個整數陣列：

```text
int[] scores;
```

此時只有陣列變數，尚未建立可存放資料的空間。使用 `new` 建立 5 個整數的空間：

```text
scores = new int[5];
```

也可以合併成一行：

```text
int[] scores = new int[5];
```

陣列建立後長度固定，不能直接改成其他長度。`int` 陣列的預設值是 `0`，`double` 陣列是 `0.0`，`boolean` 陣列是 `false`，物件型態陣列則是 `null`。

#### 範例程式

檔名：`ArrayDeclaration.java`

```java
public class ArrayDeclaration {
    public static void main(String[] args) {
        int[] scores = new int[5];
        scores[0] = 80;
        scores[1] = 75;

        String[] products = {"Keyboard", "Mouse", "Monitor"};

        System.out.println("第一筆成績：" + scores[0]);
        System.out.println("成績陣列長度：" + scores.length);
        System.out.println("第一項商品：" + products[0]);
    }
}
```

### 概念 4：索引與 `length`

陣列索引從 `0` 開始。長度為 5 的陣列，合法索引是 `0` 到 `4`。

```text
int[] scores = {80, 75, 92, 68, 88};

System.out.println(scores[0]); // 80
System.out.println(scores[4]); // 88

scores[1] = 78;
System.out.println(scores[1]); // 78
```

陣列長度使用屬性 `length`，後面沒有括號：

```text
System.out.println(scores.length);
```

最後一個元素的位置永遠是：

```text
scores[scores.length - 1]
```

#### 範例程式

檔名：`ArrayDeclaration.java`

```java
public class ArrayDeclaration {
    public static void main(String[] args) {
        int[] scores = new int[5];
        scores[0] = 80;
        scores[1] = 75;

        String[] products = {"Keyboard", "Mouse", "Monitor"};

        System.out.println("第一筆成績：" + scores[0]);
        System.out.println("成績陣列長度：" + scores.length);
        System.out.println("第一項商品：" + products[0]);
    }
}
```

### 概念 5：使用 `for` 走訪陣列

走訪 `traversal` 是依序處理每個元素。

```text
int[] scores = {80, 75, 92, 68, 88};

for (int i = 0; i < scores.length; i++) {
    System.out.println("索引 " + i + "：" + scores[i]);
}
```

條件必須是 `i < scores.length`。若寫成 `i <= scores.length`，最後一次會存取不存在的位置。

需要索引、需要修改元素或需要同時操作多個陣列時，使用一般 `for`。

#### 範例程式

檔名：`ArrayTraversal.java`

```java
public class ArrayTraversal {
    public static void main(String[] args) {
        int[] scores = {80, 75, 92, 68, 88};

        System.out.println("使用一般 for：");
        for (int i = 0; i < scores.length; i++) {
            System.out.println("索引 " + i + "：" + scores[i]);
        }

        System.out.println("使用 for-each：");
        for (int score : scores) {
            System.out.println(score);
        }
    }
}
```

### 概念 6：使用 `for-each`

只需要依序讀取資料時，可以使用增強型 `for`：

```text
int[] scores = {80, 75, 92, 68, 88};

for (int score : scores) {
    System.out.println(score);
}
```

`score` 會依序取得陣列中的值。這種寫法較簡潔，但無法直接取得索引，也不適合用來更改原陣列中的基本型態元素。

#### 範例程式

檔名：`ArrayTraversal.java`

```java
public class ArrayTraversal {
    public static void main(String[] args) {
        int[] scores = {80, 75, 92, 68, 88};

        System.out.println("使用一般 for：");
        for (int i = 0; i < scores.length; i++) {
            System.out.println("索引 " + i + "：" + scores[i]);
        }

        System.out.println("使用 for-each：");
        for (int score : scores) {
            System.out.println(score);
        }
    }
}
```

### 概念 7：使用 `Scanner` 輸入陣列

先決定資料筆數，再建立陣列：

```text
Scanner sc = new Scanner(System.in);

System.out.print("請輸入資料筆數：");
int count = sc.nextInt();

int[] values = new int[count];

for (int i = 0; i < values.length; i++) {
    System.out.print("第 " + (i + 1) + " 筆：");
    values[i] = sc.nextInt();
}
```

畫面顯示通常從第 1 筆開始，因此使用 `i + 1`；陣列索引仍然使用 `i`。

建立陣列前要檢查 `count` 是否大於 0，避免建立沒有意義或長度為負數的陣列。

#### 範例程式

檔名：`ArrayInput.java`

```java
import java.util.Scanner;

public class ArrayInput {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int count;

        do {
            System.out.print("請輸入資料筆數（1～10）：");
            count = sc.nextInt();
        } while (count < 1 || count > 10);

        int[] values = new int[count];

        for (int i = 0; i < values.length; i++) {
            System.out.print("第 " + (i + 1) + " 筆：");
            values[i] = sc.nextInt();
        }

        System.out.print("輸入結果：");
        for (int value : values) {
            System.out.print(value + " ");
        }
        System.out.println();

        sc.close();
    }
}
```

### 概念 8：總和與平均

使用累加器計算總和：

```text
int total = 0;

for (int score : scores) {
    total += score;
}

double average = (double) total / scores.length;
```

轉型為 `double` 可以避免整數除法。計算平均前也要確認陣列長度不是 0。

#### 範例程式

檔名：`ArrayStatisticsDemo.java`

```java
public class ArrayStatisticsDemo {
    public static void main(String[] args) {
        int[] scores = {80, 75, 92, 68, 88};
        int total = 0;
        int max = scores[0];
        int min = scores[0];
        int passCount = 0;

        for (int score : scores) {
            total += score;

            if (score > max) {
                max = score;
            }
            if (score < min) {
                min = score;
            }
            if (score >= 60) {
                passCount++;
            }
        }

        double average = (double) total / scores.length;

        System.out.println("總分：" + total);
        System.out.printf("平均：%.2f%n", average);
        System.out.println("最高分：" + max);
        System.out.println("最低分：" + min);
        System.out.println("及格人數：" + passCount);
    }
}
```

### 概念 9：最大值與最小值

最大值與最小值應以第一個元素作為初始值：

```text
int max = scores[0];
int min = scores[0];

for (int i = 1; i < scores.length; i++) {
    if (scores[i] > max) {
        max = scores[i];
    }

    if (scores[i] < min) {
        min = scores[i];

}
```

### 概念 11：循序搜尋

循序搜尋 `sequential search` 從第一個元素開始逐一比對：

```text
public static int findIndex(int[] values, int target) {
    for (int i = 0; i < values.length; i++) {
        if (values[i] == target) {
            return i;
        }
    }

    return -1;
}
```

找到時回傳索引，找不到時回傳 `-1`。因為合法索引不會是負數，所以 `-1` 適合表示「不存在」。

#### 範例程式

檔名：`SequentialSearch.java`

```java
public class SequentialSearch {
    public static int findIndex(int[] values, int target) {
        for (int i = 0; i < values.length; i++) {
            if (values[i] == target) {
                return i;
            }
        }
        return -1;
    }

    public static void main(String[] args) {
        int[] values = {12, 25, 18, 30, 25};
        int target = 25;
        int index = findIndex(values, target);

        if (index >= 0) {
            System.out.println(target + " 第一次出現在索引 " + index);
        } else {
            System.out.println("找不到 " + target);
        }
    }
}
```

### 概念 12：陣列作為方法參數

陣列可以傳入方法，讓每個方法只負責一項工作：

```text
public static int calculateTotal(int[] values) {
    int total = 0;

    for (int value : values) {
        total += value;
    }

    return total;
}
```

呼叫方式：

```text
int total = calculateTotal(scores);
```

方法接收到的是同一個陣列的參考。若方法內修改 `values[0]`，原本陣列的第一個元素也會改變。

#### 範例程式

檔名：`ArrayMethodDemo.java`

```java
public class ArrayMethodDemo {
    public static int calculateTotal(int[] values) {
        int total = 0;
        for (int value : values) {
            total += value;
        }
        return total;
    }

    public static void addBonus(int[] values, int bonus) {
        for (int i = 0; i < values.length; i++) {
            values[i] += bonus;
        }
    }

    public static void printArray(int[] values) {
        for (int value : values) {
            System.out.print(value + " ");
        }
        System.out.println();
    }

    public static void main(String[] args) {
        int[] scores = {70, 75, 80};

        System.out.println("原始資料：");
        printArray(scores);
        System.out.println("總和：" + calculateTotal(scores));

        addBonus(scores, 5);
        System.out.println("加分後：");
        printArray(scores);
    }
}
```

### 概念 13：複製陣列

以下寫法不會建立新陣列：

```text
int[] copy = scores;
```

`copy` 與 `scores` 會指向同一個陣列。若要建立獨立副本，可以使用：

```text
import java.util.Arrays;

int[] copy = Arrays.copyOf(scores, scores.length);
```

修改 `copy` 後，不會影響原本的 `scores`。

#### 範例程式

檔名：`ArrayCopyDemo.java`

```java
import java.util.Arrays;

public class ArrayCopyDemo {
    public static void main(String[] args) {
        int[] original = {10, 20, 30};
        int[] copy = Arrays.copyOf(original, original.length);

        copy[0] = 99;

        System.out.println("原陣列：" + Arrays.toString(original));
        System.out.println("副本：" + Arrays.toString(copy));
    }
}
```

### 概念 14：二維陣列

二維陣列適合表示表格。例如 3 項商品、4 天銷售量：

```text
int[][] sales = {
    {5, 8, 6, 7},
    {3, 4, 5, 6},
    {10, 12, 9, 11}
};
```

`sales[0][2]` 表示第 1 列、第 3 欄，值為 6。

`sales.length` 是列數，`sales[row].length` 是指定列的欄數。

#### 範例程式

檔名：`TwoDimensionalArray.java`

```java
public class TwoDimensionalArray {
    public static void main(String[] args) {
        int[][] matrix = {
            {10, 20, 30},
            {40, 50, 60}
        };

        for (int row = 0; row < matrix.length; row++) {
            for (int column = 0; column < matrix[row].length; column++) {
                System.out.printf("%4d", matrix[row][column]);
            }
            System.out.println();
        }
    }
}
```

### 概念 15：巢狀迴圈走訪二維陣列

```text
for (int row = 0; row < sales.length; row++) {
    for (int column = 0; column < sales[row].length; column++) {
        System.out.print(sales[row][column] + " ");
    }
    System.out.println();
}
```

外層迴圈控制列，內層迴圈控制欄。可以在每一列建立累加器，計算各商品的銷售總量。

#### 範例程式

檔名：`SalesMatrixDemo.java`

```java
public class SalesMatrixDemo {
    public static void main(String[] args) {
        int[][] sales = {
            {5, 8, 6, 7},
            {3, 4, 5, 6},
            {10, 12, 9, 11}
        };

        for (int product = 0; product < sales.length; product++) {
            int productTotal = 0;
            for (int day = 0; day < sales[product].length; day++) {
                productTotal += sales[product][day];
            }
            System.out.println("商品 " + (product + 1)
                    + " 總銷售量：" + productTotal);
        }

        for (int day = 0; day < sales[0].length; day++) {
            int dayTotal = 0;
            for (int product = 0; product < sales.length; product++) {
                dayTotal += sales[product][day];
            }
            System.out.println("第 " + (day + 1)
                    + " 天總銷售量：" + dayTotal);
        }
    }
}
```

## 常見錯誤與診斷

| 常見錯誤 | 原因 | 修正方式 |
|---|---|---|
| `ArrayIndexOutOfBoundsException` | 使用了不存在的索引 | 確認索引範圍是 0 到 `length - 1` |
| 迴圈多執行一次 | 使用 `i <= array.length` | 改成 `i < array.length` |
| 平均值沒有小數 | 使用整數除法 | 將總和轉型為 `double` |
| 最大值在負數資料中錯誤 | 最大值初始為 0 | 使用第一個元素作為初始值 |
| 修改副本時原陣列一起改變 | 兩個變數指向同一陣列 | 使用 `Arrays.copyOf` |
| 二維陣列列欄顛倒 | 混用 `row` 與 `column` | 外層控制列，內層控制欄 |
| 找不到資料卻存取 `-1` | 未先判斷搜尋結果 | 只有索引大於等於 0 才能存取陣列 |
