### 概念 2：`String` 基本概念

`String` 用來保存文字：

```text
String productName = "Keyboard";
String message = "Welcome to Java";
```

字串由多個字元組成，每個字元也有從 0 開始的索引。

`String` 是物件型態，不是基本資料型態。字串建立後內容不可直接修改，稱為 immutable。呼叫轉換方法時會得到新的字串：

```text
String name = "keyboard";
String upperName = name.toUpperCase();

System.out.println(name);      // keyboard
System.out.println(upperName); // KEYBOARD
```

#### 範例程式

檔名：`StringBasics.java`

```java
public class StringBasics {
    public static void main(String[] args) {
        String original = "  Java Programming  ";
        String text = original.trim();

        System.out.println("原始文字：" + original);
        System.out.println("移除前後空白：" + text);
        System.out.println("長度：" + text.length());
        System.out.println("第一個字元：" + text.charAt(0));
        System.out.println("最後字元："
                + text.charAt(text.length() - 1));
        System.out.println("前四個字元：" + text.substring(0, 4));
        System.out.println("從索引 5 開始：" + text.substring(5));
        System.out.println("大寫：" + text.toUpperCase());
        System.out.println("小寫：" + text.toLowerCase());
        System.out.println("原始字串仍未改變：" + original);
    }
}
```

### 概念 3：`length()` 與 `charAt()`

字串長度使用方法 `length()`：

```text
String text = "Java";
System.out.println(text.length()); // 4
```

取得指定位置字元：

```text
char first = text.charAt(0);
char last = text.charAt(text.length() - 1);
```

字串最後一個合法索引也是 `length() - 1`。空字串長度為 0，不能呼叫 `charAt(0)`。

### 概念 4：`substring()`

`substring(start)` 會取得從 `start` 到結尾的文字：

```text
String code = "TKU-0715";
System.out.println(code.substring(4)); // 0715
```

`substring(start, end)` 包含起始位置，但不包含結束位置：

```text
System.out.println(code.substring(0, 3)); // TKU
```

可以記成 `start <= index < end`。

### 概念 5：搜尋文字

`indexOf()` 回傳第一次出現的位置，找不到回傳 `-1`：

```text
String text = "Java Programming";
int index = text.indexOf("Program");
System.out.println(index); // 5
```

`contains()` 直接回傳 `boolean`：

```text
boolean found = text.contains("Java");
```

兩者都會區分英文大小寫。

#### 範例程式

檔名：`StringSearchDemo.java`

```java
public class StringSearchDemo {
    public static void main(String[] args) {
        String text = "Java Programming and Data Structures";
        String keyword = "data";

        String normalizedText = text.toLowerCase();
        String normalizedKeyword = keyword.toLowerCase();

        System.out.println("是否包含："
                + normalizedText.contains(normalizedKeyword));
        System.out.println("第一次出現位置："
                + normalizedText.indexOf(normalizedKeyword));
    }
}
```

```text
String input = "  Keyboard  ";
```

### 概念 7：`==` 與 `equals()`

比較字串內容應使用 `equals()`：

```text
String input = new String("Java");

System.out.println(input == "Java");       // 不應依賴此結果
System.out.println(input.equals("Java"));  // true
```

`==` 比較兩個變數是否指向同一個物件；`equals()` 比較文字內容。

忽略大小寫時使用：

```text
input.equalsIgnoreCase("java")
```

如果變數可能是 `null`，可將確定不是 `null` 的字串寫在前面：

```text
"Java".equals(input)
```

#### 範例程式

檔名：`StringCompareDemo.java`

```java
public class StringCompareDemo {
    public static void main(String[] args) {
        String first = new String("Java");
        String second = "Java";

        System.out.println(first == second);
        System.out.println(first.equals(second));
        System.out.println(first.equalsIgnoreCase("java"));
    }
}
```

### 概念 8：使用 `split()` 拆解文字

`split()` 會依分隔符號拆成 `String[]`：

```text
String record = "Keyboard,890,12";
String[] parts = record.split(",");

System.out.println(parts[0]); // Keyboard
System.out.println(parts[1]); // 890
System.out.println(parts[2]); // 12
```

使用前應先檢查欄位數：

```text
if (parts.length != 3) {
    System.out.println("資料格式錯誤");
}
```

分隔符號是 regular expression。若要用句點切割，必須寫成 `split("\\.")`。

#### 範例程式

檔名：`StringSplitDemo.java`

```java
public class StringSplitDemo {
    public static void main(String[] args) {
        String record = " Keyboard , 890 , 12 ";
        String[] parts = record.split(",");

        if (parts.length == 3) {
            String name = parts[0].trim();
            int price = Integer.parseInt(parts[1].trim());
            int stock = Integer.parseInt(parts[2].trim());

            System.out.println("名稱：" + name);
            System.out.println("價格：" + price);
            System.out.println("庫存：" + stock);
        } else {
            System.out.println("資料格式錯誤");
        }
    }
}
```

### 概念 9：字串轉數值

`split()` 得到的每個欄位仍是字串。轉為整數：

```text
int price = Integer.parseInt(parts[1].trim());
int stock = Integer.parseInt(parts[2].trim());
```

若內容不是合法整數，會發生 `NumberFormatException`。可以使用 `try-catch` 處理：

```text
try {
    int value = Integer.parseInt(input);
    System.out.println("數值：" + value);
} catch (NumberFormatException e) {
    System.out.println("請輸入整數");
}
```

`try` 放可能發生錯誤的程式，`catch` 處理指定例外。

#### 範例程式

檔名：`NumberParsingDemo.java`

```java
import java.util.Scanner;

public class NumberParsingDemo {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("請輸入整數：");
        String input = sc.nextLine();

        try {
            int value = Integer.parseInt(input.trim());
            System.out.println("兩倍為：" + value * 2);
        } catch (NumberFormatException e) {
            System.out.println("格式錯誤，必須輸入整數");
        }

        sc.close();
    }
}
```

### 概念 10：`Scanner` 的換行問題

`nextInt()` 不會讀走輸入後的換行。若接著使用 `nextLine()`，可能立即得到空字串：

```text
int age = sc.nextInt();
sc.nextLine(); // 讀走剩餘換行
String name = sc.nextLine();
```

另一種做法是所有輸入都使用 `nextLine()`，需要數字時再用 `Integer.parseInt()` 轉換。

#### 範例程式

檔名：`ScannerLineDemo.java`

```java
import java.util.Scanner;

public class ScannerLineDemo {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("請輸入年齡：");
        int age = sc.nextInt();
        sc.nextLine();

        System.out.print("請輸入姓名：");
        String name = sc.nextLine();

        System.out.println("姓名：" + name);
        System.out.println("年齡：" + age);

        sc.close();
    }
}
```

### 概念 11：字串陣列與搜尋

```text
String[] names = {"Keyboard", "Mouse", "Monitor"};

public static int findProduct(String[] names, String keyword) {
    for (int i = 0; i < names.length; i++) {
        if (names[i].equalsIgnoreCase(keyword.trim())) {
            return i;
        }
    }
    return -1;
}
```

若要做部分比對，可以先轉成小寫：

```text
if (names[i].toLowerCase().contains(keyword.toLowerCase())) {
    // 找到部分相符的商品
}
```

#### 範例程式

檔名：`StringArrayDemo.java`

```java
public class StringArrayDemo {
    public static int findExact(String[] values, String keyword) {
        for (int i = 0; i < values.length; i++) {
            if (values[i].equalsIgnoreCase(keyword.trim())) {
                return i;
            }
        }
        return -1;
    }

    public static void printPartial(String[] values, String keyword) {
        String normalized = keyword.trim().toLowerCase();
        boolean found = false;

        for (String value : values) {
            if (value.toLowerCase().contains(normalized)) {
                System.out.println(value);
                found = true;
            }
        }

        if (!found) {
            System.out.println("找不到相符資料");
        }
    }

    public static void main(String[] args) {
        String[] products = {
            "Keyboard", "Mouse", "Monitor", "USB Cable", "Headset"
        };

        int index = findExact(products, " mouse ");
        System.out.println("完整搜尋索引：" + index);
        System.out.println("部分搜尋結果：");
        printPartial(products, "o");
    }
}
```

### 概念 12：`StringBuilder`

在迴圈中重複組合大量文字時，可使用 `StringBuilder`：

```text
StringBuilder result = new StringBuilder();

for (String name : names) {
    result.append(name).append("\n");
}

System.out.print(result.toString());
```

常用方法包括 `append()`、`insert()`、`delete()` 與 `reverse()`。

#### 範例程式

檔名：`StringBuilderDemo.java`

```java
public class StringBuilderDemo {
    public static void main(String[] args) {
        String[] names = {"Keyboard", "Mouse", "Monitor"};
        StringBuilder report = new StringBuilder();

        report.append("商品清單\n");
        report.append("--------\n");

        for (int i = 0; i < names.length; i++) {
            report.append(i + 1)
                    .append(". ")
                    .append(names[i])
                    .append("\n");
        }

        System.out.print(report.toString());
    }
}
```

### 概念 13：三種程式錯誤

#### 編譯錯誤 compile-time error

程式不符合 Java 語法，無法產生可執行結果。例如少分號、變數不存在或型態不相容。

#### 執行錯誤 runtime error

程式可以編譯，但執行時發生例外。例如陣列越界、除以 0 或文字轉數字失敗。

#### 邏輯錯誤 logic error

程式可以執行，也沒有例外，但結果錯誤。例如平均值使用整數除法、迴圈少處理最後一筆資料。

#### 範例程式

檔名：`DebuggingFixedDemo.java`

```java
public class DebuggingFixedDemo {
    public static void main(String[] args) {
        String[] names = {"Keyboard", "Mouse", "Monitor"};
        int[] prices = {890, 490, 5200};
        int total = 0;

        for (int i = 0; i < prices.length; i++) {
            total += prices[i];
            System.out.println(names[i] + "：" + prices[i]);
        }

        double average = (double) total / prices.length;
        System.out.printf("平均價格：%.2f%n", average);

        String target = new String("Mouse");
        if (target.equals(names[1])) {
            System.out.println("找到 Mouse");
        }
    }
}
```

### 概念 14：使用 VS Code Debugger

1. 在程式行號左側點一下，建立紅色 breakpoint。
2. 按 `F5` 或選擇 Run and Debug。
3. 程式停在 breakpoint 後，查看 Variables。
4. 使用 Step Over 執行下一行。
5. 使用 Step Into 進入自訂方法。
6. 使用 Continue 執行到下一個 breakpoint。
7. 觀察索引、累加器與陣列內容是否符合預期。

適合設 breakpoint 的位置：

- 迴圈開始處。
- 條件判斷前。
- 方法呼叫前後。
- 結果第一次變得不正確的位置。

#### 範例程式

檔名：`DebuggingFixedDemo.java`

```java
public class DebuggingFixedDemo {
    public static void main(String[] args) {
        String[] names = {"Keyboard", "Mouse", "Monitor"};
```

### 概念 15：設計測試案例

測試不能只輸入一組順利資料。至少包含：

| 類型 | 說明 | 商品搜尋範例 |
|---|---|---|
| 正常值 | 一般合法資料 | 搜尋 `Mouse` |
| 邊界值 | 合法範圍的最小或最大值 | 購買全部剩餘庫存 |
| 錯誤值 | 不符合規則的資料 | 輸入負數數量 |
| 空值 | 沒有有效內容 | 搜尋空字串 |
| 不存在 | 格式合法但找不到 | 搜尋 `Camera` |
| 格式錯誤 | 欄位或數字格式錯誤 | `Mouse,abc,10` |

每個測試案例應記錄輸入、預期結果與實際結果。

#### 範例程式

檔名：`ManualTestCases.java`

```java
public class ManualTestCases {
    public static boolean isValidScore(int score) {
        return score >= 0 && score <= 100;
    }

    public static void runTest(int input, boolean expected) {
        boolean actual = isValidScore(input);
        String result = actual == expected ? "通過" : "失敗";

        System.out.println("輸入：" + input
                + "，預期：" + expected
                + "，實際：" + actual
                + "，結果：" + result);
    }

    public static void main(String[] args) {
        runTest(80, true);
        runTest(0, true);
        runTest(100, true);
        runTest(-1, false);
        runTest(101, false);
    }
}
```

## 常見錯誤與診斷

| 常見錯誤 | 原因 | 修正方式 |
|---|---|---|
| 相同文字比較結果為 false | 使用 `==` | 改用 `equals()` 或 `equalsIgnoreCase()` |
| `StringIndexOutOfBoundsException` | 字元或子字串索引超出範圍 | 先檢查 `length()` |
| `NumberFormatException` | 文字不是合法數字 | 使用 `try-catch` 並顯示錯誤 |
| `split()` 後陣列越界 | 欄位數不足 | 先檢查 `parts.length` |
| `nextLine()` 立即得到空字串 | 前一次 `nextInt()` 留下換行 | 多執行一次 `nextLine()` |
| 部分搜尋找不到大小寫不同資料 | 直接使用 `contains()` | 兩邊先轉成相同大小寫 |
| 除錯時不知道從哪裡開始 | breakpoint 設在無關位置 | 從結果第一次異常的位置往前追蹤 |
| 只測試一組正常資料 | 測試範圍不足 | 補上邊界、錯誤、空值與不存在案例 |
