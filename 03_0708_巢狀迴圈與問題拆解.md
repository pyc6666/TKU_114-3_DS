### 概念 3：累加器 accumulator

#### 概念說明

累加器是用來把多次計算結果加總起來的變數。常見名稱：

```java
int total = 0;
double totalAmount = 0.0;
```

每次有新的數值，就加到累加器中：

```java
total = total + score;
```

也可以寫成：

```java
total += score;
```

兩種寫法意思相同。

#### 範例程式

檔名：`AccumulatorDemo.java`

```java
public class AccumulatorDemo {
    public static void main(String[] args) {
        int total = 0;

        total += 80;
        total += 90;
        total += 75;

        System.out.println("Total: " + total);
    }
}
```

### 概念 5：最大值與最小值

#### 概念說明

如果要找最大值，可以先把第一筆資料當作目前最大值，再逐筆比較。

基本想法：

```java
int max = firstValue;

if (newValue > max) {
    max = newValue;
}
```

找最小值也類似：

```java
int min = firstValue;

if (newValue < min) {
    min = newValue;
}
```

#### 範例程式

檔名：`MaxMinDemo.java`

```java
public class MaxMinDemo {
    public static void main(String[] args) {
        int score1 = 80;
        int score2 = 55;
        int score3 = 92;

        int max = score1;
        int min = score1;

        if (score2 > max) {
            max = score2;
        }

        if (score3 > max) {
            max = score3;
        }

        if (score2 < min) {
            min = score2;
        }

        if (score3 < min) {
            min = score3;
        }

        System.out.println("Max: " + max);
        System.out.println("Min: " + min);
    }
}
```

### 概念 6：巢狀迴圈 nested loop

#### 概念說明

巢狀迴圈就是「迴圈裡面還有迴圈」。常用在表格、座標、矩陣、星號圖形。

基本格式：

```java
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        System.out.println("i=" + i + ", j=" + j);
    }
}
```

外層迴圈跑一次，內層迴圈會完整跑完一輪。

#### 範例程式：九九乘法表

檔名：`MultiplicationTable.java`

```java
public class MultiplicationTable {
    public static void main(String[] args) {
        for (int i = 1; i <= 9; i++) {
            for (int j = 1; j <= 9; j++) {
                System.out.print(i + "x" + j + "=" + (i * j) + "\t");
            }
            System.out.println();
        }
    }
}
```

### 概念 8：輸入驗證 input validation

#### 概念說明

輸入驗證是指：如果使用者輸入不合理的資料，就要求重新輸入。

例如分數應該在 0 到 100 之間。如果輸入 -10 或 150，程式不應直接接受。

常見寫法：

```java
while (score < 0 || score > 100) {
    System.out.print("請重新輸入：");
    score = sc.nextInt();
}
```

#### 範例程式

檔名：`InputValidation.java`

```java
import java.util.Scanner;

public class InputValidation {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("請輸入分數（0-100）：");
        int score = sc.nextInt();

        while (score < 0 || score > 100) {
            System.out.print("分數不合法，請重新輸入（0-100）：");
            score = sc.nextInt();
        }

        System.out.println("Score: " + score);

        sc.close();
    }
}
```

### 概念 9：選單式程式 menu-driven program

#### 概念說明

選單式程式會重複顯示功能選項，讓使用者輸入選項。通常使用：

- `while` 控制是否繼續執行。
- `switch` 判斷使用者選擇哪個功能。

基本結構：

```java
while (option != 0) {
    顯示選單
    讀取 option

    switch (option) {
        case 1:
            執行功能 1
            break;
        case 0:
            離開
            break;
        default:
            顯示錯誤選項
    }
}
```

#### 範例程式

檔名：`SimpleMenu.java`

```java
import java.util.Scanner;

public class SimpleMenu {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int option = -1;

        while (option != 0) {
            System.out.println("=== Menu ===");
            System.out.println("1. Say hello");
            System.out.println("2. Say Java");
            System.out.println("0. Exit");
            System.out.print("請輸入選項：");
            option = sc.nextInt();

            switch (option) {
                case 1:
                    System.out.println("Hello");
                    break;
                case 2:
                    System.out.println("Java");
                    break;
                case 0:
                    System.out.println("Bye");
                    break;
                default:
                    System.out.println("Unknown option");
            }
        }

        sc.close();
    }
}
```

## 常見錯誤與診斷

| 常見錯誤                    | 可能原因              | 診斷方式                    | 修正方式                               |
| --------------------------- | --------------------- | --------------------------- | -------------------------------------- |
| 總金額沒有增加              | 忘記使用累加器        | 每次點餐後 total 沒變       | 使用`totalAmount += subtotal`          |
| 數量只記最後一次            | 沒有累加總數量        | 結帳時只顯示最後一次數量    | 使用`totalItems += quantity`           |
| 無限迴圈                    | `while` 條件沒有改變  | 選 0 後還是無法離開         | 確認 option 有被重新輸入或條件有改變   |
| 巢狀迴圈輸出格式錯誤        | 忘記在外層迴圈換行    | 圖形全部擠在同一行          | 內層迴圈結束後加`System.out.println()` |
| 最大最小值錯誤              | 初始值設定不合理      | 結果永遠是 0 或固定值       | 用第一筆資料當初始 max / min           |
| 數量可以輸入 0 或負數       | 沒有做輸入驗證        | 小計變成 0 或負數           | 用`while (quantity <= 0)` 要求重輸     |
| `switch` 忘記 `break`       | case 執行後繼續往下跑 | 一次輸出多個功能結果        | 每個`case` 結尾加入 `break`            |
| iClass 只貼 repository 首頁 | 沒有貼到當天資料夾    | 連結打開後看不到`0708` 內容 | 貼上`.../tree/main/0708` 形式的連結    |
