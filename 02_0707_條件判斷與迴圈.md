# 7/7 教材：條件判斷與迴圈

## 單元名稱

Java 條件判斷、邏輯運算、`switch`、`for` 迴圈與 `while` 迴圈

## 課程定位

7/2 已經完成 Java 開發環境、GitHub repository、基本輸入輸出、變數、資料型態與簡單運算。本單元開始讓程式具備「判斷」與「重複」能力。

只會計算數字的程式還不夠完整。實際程式通常需要根據資料做不同決定，例如：

- 分數是否及格？
- BMI 屬於哪一個區間？
- 使用者選擇哪一個功能？
- 是否要繼續輸入下一筆資料？

本單元會把 7/2 的 `HealthCalculator.java` 與 `ScoreReport.java` 延伸成更接近小系統的程式。

### 概念 2：`boolean` 與比較運算子

#### 概念說明

`boolean` 是 Java 中用來表示「成立」或「不成立」的資料型態，只有兩種值：

```java
true
false
```

條件判斷通常會得到 `boolean` 結果。例如：

```java
score >= 60
```

如果 `score` 是 80，結果是 `true`。如果 `score` 是 50，結果是 `false`。

常用比較運算子：

| 運算子 | 意義 | 範例 |
|---|---|---|
| `>` | 大於 | `score > 60` |
| `<` | 小於 | `age < 18` |
| `>=` | 大於或等於 | `score >= 60` |
| `<=` | 小於或等於 | `bmi <= 24` |
| `==` | 等於 | `option == 1` |
| `!=` | 不等於 | `option != 0` |

注意：判斷兩個值是否相等時使用 `==`，不是 `=`。  
`=` 是指定值，例如 `score = 80`。  
`==` 是比較，例如 `score == 80`。

#### 範例程式

檔名：`BooleanDemo.java`

```java
public class BooleanDemo {
    public static void main(String[] args) {
        int score = 75;
        boolean pass = score >= 60;

        System.out.println("Score: " + score);
        System.out.println("Pass: " + pass);
    }
}
```

### 概念 3：邏輯運算子 `&&`、`||`、`!`

#### 概念說明

有時候一個條件不夠，需要把多個條件組合起來。Java 常用三個邏輯運算子：

| 運算子 | 意義 | 成立條件 |
|---|---|---|
| `&&` | and，而且 | 兩邊都成立才是 `true` |
| `||` | or，或者 | 其中一邊成立就是 `true` |
| `!` | not，反向 | 把 `true` 變 `false`，把 `false` 變 `true` |

例子：

```java
score >= 60 && attendance >= 80
```

代表分數至少 60，而且出席率至少 80。

```java
option == 1 || option == 2
```

代表選項是 1 或 2 都可以。

#### 範例程式

檔名：`LogicDemo.java`

```java
public class LogicDemo {
    public static void main(String[] args) {
        int score = 75;
        int attendance = 90;

        boolean passCourse = score >= 60 && attendance >= 80;

        System.out.println("Score: " + score);
        System.out.println("Attendance: " + attendance);
        System.out.println("Pass course: " + passCourse);
    }
}
```


### 概念 4：`if`

#### 概念說明

`if` 用來表示「如果條件成立，就執行某段程式」。

基本格式：

```java
if (條件) {
    條件成立時執行的程式
}
```

如果條件不成立，大括號裡的程式就會被跳過。

#### 範例程式

檔名：`PassCheck.java`

```java
public class PassCheck {
    public static void main(String[] args) {
        int score = 80;

        System.out.println("Score: " + score);

        if (score >= 60) {
            System.out.println("Pass");
        }
    }
}
```

### 概念 5：`if else`

#### 概念說明

`if else` 用在二選一情境。

基本格式：

```java
if (條件) {
    條件成立時執行
} else {
    條件不成立時執行
}
```

例如成績達 60 分就是及格，否則是不及格。

#### 範例程式

檔名：`PassFail.java`

```java
public class PassFail {
    public static void main(String[] args) {
        int score = 55;

        if (score >= 60) {
            System.out.println("Pass");
        } else {
            System.out.println("Fail");
        }
    }
}
```

### 概念 6：`if else if else`

#### 概念說明

當條件不是二選一，而是多個區間時，可以使用 `if else if else`。

基本格式：

```java
if (條件 1) {
    條件 1 成立時執行
} else if (條件 2) {
    條件 2 成立時執行
} else {
    以上條件都不成立時執行
}
```

判斷會由上往下進行，只要某一個條件成立，後面的條件就不會再判斷。

#### 範例程式：成績等第

檔名：`GradeLevel.java`

```java
public class GradeLevel {
    public static void main(String[] args) {
        int score = 85;

        if (score >= 90) {
            System.out.println("Grade: A");
        } else if (score >= 80) {
            System.out.println("Grade: B");
        } else if (score >= 70) {
            System.out.println("Grade: C");
        } else if (score >= 60) {
            System.out.println("Grade: D");
        } else {
            System.out.println("Grade: F");
        }
    }
}
```

#### 範例程式：BMI 分級

檔名：`BmiLevel.java`

```java
public class BmiLevel {
    public static void main(String[] args) {
        double bmi = 22.5;

        if (bmi < 18.5) {
            System.out.println("BMI level: Underweight");
        } else if (bmi < 24) {
            System.out.println("BMI level: Normal");
        } else if (bmi < 27) {
            System.out.println("BMI level: Overweight");
        } else {
            System.out.println("BMI level: Obese");
        }
    }
}
```

### 概念 7：`switch`

#### 概念說明

`switch` 適合處理「固定選項」的情境，例如選單。

基本格式：

```java
switch (變數) {
    case 值1:
        選到值1時執行
        break;
    case 值2:
        選到值2時執行
        break;
    default:
        其他情況執行
}
```

`break` 的作用是離開 `switch`。如果忘記 `break`，程式可能會繼續往下一個 `case` 執行。

#### 範例程式

檔名：`MenuDemo.java`

```java
public class MenuDemo {
    public static void main(String[] args) {
        int option = 2;

        switch (option) {
            case 1:
                System.out.println("Show BMI");
                break;
            case 2:
                System.out.println("Show score average");
                break;
            case 3:
                System.out.println("Show grade level");
                break;
            default:
                System.out.println("Unknown option");
        }
    }
}
```

### 概念 8：`for` 迴圈

#### 概念說明

`for` 適合用在「已經知道要重複幾次」的情境。

基本格式：

```java
for (初始值; 條件; 更新) {
    重複執行的程式
}
```

常見例子：

```java
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}
```

意思是：

1. 從 `i = 1` 開始。
2. 只要 `i <= 5` 成立，就執行大括號內程式。
3. 每次執行完後，做 `i++`，讓 `i` 增加 1。

#### 範例程式

檔名：`ForLoopDemo.java`

```java
public class ForLoopDemo {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Round " + i);
        }
    }
}
```

#### 課堂實作題

建立 `SumOneToTen.java`，使用 `for` 迴圈計算 1 到 10 的總和。

預期輸出：

```text
Sum: 55
```

### 概念 9：`while` 迴圈

#### 概念說明

`while` 適合用在「只要條件成立，就繼續重複」的情境。

基本格式：

```java
while (條件) {
    重複執行的程式
}
```

`while` 要特別注意條件是否會改變。如果條件永遠成立，程式就會變成無限迴圈。

#### 範例程式

檔名：`WhileCountDown.java`

```java
public class WhileCountDown {
    public static void main(String[] args) {
        int count = 3;

        while (count > 0) {
            System.out.println(count);
            count--;
        }

        System.out.println("Start");
    }
}
```

#### 課堂實作題

建立 `WhileInputDemo.java`，使用 `Scanner` 讀取使用者輸入的整數，只要輸入不是 0，就持續輸出該數字。

### 概念 10：`break` 與 `continue`

#### 概念說明

`break` 與 `continue` 都可以改變迴圈流程。

- `break`：直接離開迴圈。
- `continue`：跳過本次迴圈剩下的程式，直接進入下一次迴圈。

`break` 常用在選單中的離開功能。`continue` 常用在略過不需要處理的資料。

#### 範例程式

檔名：`BreakContinueDemo.java`

```java
public class BreakContinueDemo {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            if (i == 3) {
                continue;
            }

            if (i == 5) {
                break;
            }

            System.out.println(i);
        }
    }
}
```

執行結果：

```text
1
2
4
```

## 常見錯誤與診斷

| 常見錯誤 | 可能原因 | 診斷方式 | 修正方式 |
|---|---|---|---|
| 使用 `=` 判斷相等 | `=` 是指定值，不是比較 | 條件判斷出現錯誤或無法編譯 | 比較相等時使用 `==` |
| 字串使用 `==` 比較 | `String` 應使用 `.equals()` | 輸入看起來一樣但判斷失敗 | 使用 `text.equals("y")` |
| `if else if` 順序錯誤 | 大範圍條件放太前面 | 等第或 BMI 分級不正確 | 由高到低或由小到大安排條件 |
| 忘記 `break` | `switch` 繼續執行下一個 `case` | 輸出多個不該出現的結果 | 每個 `case` 結尾加入 `break` |
| 無限迴圈 | `while` 條件永遠成立 | 程式一直執行不停止 | 確認迴圈內有更新條件或離開機制 |
| `Scanner` 讀取順序混亂 | `nextInt()` 後接 `nextLine()` | 字串輸入被跳過 | 讀完數字後補一個 `sc.nextLine()` 或調整輸入順序 |
| GitHub 看不到 `0707` | 還沒 push 或資料夾沒有檔案 | GitHub 頁面沒有 `0707` | commit 後 push，並確認資料夾中有 `.java` 檔案 |
| iClass 只貼 repository 首頁 | 沒有貼到當天資料夾 | 連結打開後看不到 `0707` 內容 | 貼上 `.../tree/main/0707` 形式的連結 |
